# ProofBuddy

ProofBuddy is an AI writing integrity assistant for educators. Instead of making accusations or giving a simple AI-detection score, it provides evidence-based review signals such as prompt alignment, writing consistency, citation issues, and suggested follow-up questions.

## Hackathon MVP

The MVP will ideally include:

- Educator submission review form
- AI policy selector
- Writing integrity report dashboard
- Prompt alignment review
- Writing consistency review
- Citation audit
- Follow-up question generation
